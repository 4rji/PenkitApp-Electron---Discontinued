Surf in Stealth Mode: Unlock the Power of Proxychains!

Proxychains es una herramienta para enrutar tu tráfico a través de una serie de servidores proxy, brindando anonimato adicional y la capacidad de eludir restricciones de red. Ideal para pentesters y entusiastas de la privacidad que buscan ocultar su actividad online.

    Modifica la configuración para modo dinámico y agrega los proxies

    sudo nano /etc/proxychains.conf

#uncomment only

random_chain
# Proxy DNS requests - no leak for DNS data
proxy_dns

# Some timeouts in milliseconds
tcp_read_time_out 15000
tcp_connect_time_out 8000

#add squid servers:
http 192.168.88.83 3128
http 192.168.2.2 3128

#add some ssh -d 1080 servers
socks4 192.168.88.83 1080
socks4 192.168.200.228 1080
