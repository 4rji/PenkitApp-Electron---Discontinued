iperf3 Speed Test

Context

This script performs a speed test using iperf3, measuring both upload and download speeds to a specified server.

Usage: veloint [SERVER_IP]

Functionality


1. Prompts the user to start the test.
2. Runs an upload test to the specified server.
3. Runs a download test using the --reverse flag.
4. Extracts the bitrate from the results.
5. Displays a summary of upload and download speeds.

Example Output


Upload Test:
  Bitrate (sender):   850 Mbps
  Bitrate (receiver): 840 Mbps

Download Test:
  Bitrate (sender):   900 Mbps
  Bitrate (receiver): 890 Mbps

Requirements


Ensure that iperf3 is installed on your system. If not, install it using:

sudo apt install iperf3   # Debian/Ubuntu
brew install iperf3       # macOS
sudo dnf install iperf3   # Fedora

Notes


If you experience connection issues, ensure that the specified server is running an iperf3 instance in server mode.
